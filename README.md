# 🧟 营救女孩 - 僵尸围城

颜色匹配塔防消除H5小游戏，对标抖音小游戏《营救女孩》。

## 🎮 在线试玩

👉 https://timmmmmo.github.io/rescue-girl-zombie/

## 🎯 玩法

- 僵尸从右侧逼近女孩，身上带有不同颜色的能量球
- 点击对应颜色的大炮发射炮弹
- 匹配成功 → 消灭僵尸能量球 → 击退僵尸
- 女孩血量归零则失败

## ✨ 特性

- 五色大炮系统（红/蓝/绿/橙/紫）
- Combo连击系统
- 粒子特效 + 屏幕震动
- 关卡递进难度
- 触屏+鼠标双操控
- 单文件H5，零依赖

## 📦 技术

纯 HTML5 + Canvas + JavaScript，单文件即可运行。
